This is a boilerplate for a [React](https://reactjs.org) project using [Hugo](http://gohugo.io/) as a [build tool](https://gohugo.io/hugo-pipes/js/).

### Requirements

-   [Hugo](http://gohugo.io/)
-   [Node](https://nodejs.org/)

### Start

```
npm install
```

### Development

```
npm start
```

### Build

```
npm run build
```
